to run

To run our website, please go to Tra_well/application/controllers, and run the home.php

Homepage, Login Page

On homepage, click the title on top-left corner; then the page can jump back to the homepage.

If click on the login button on top-right corner, the page will jump login page. On login page, the user can use the left section to login and use the right section to register an account. The login and register buttons are templetes, and we will work on them later.

Beside login button, there is a search bar and search button used for searching.

At the bottom of the home page, there are five circles which link to five different cities(right now only Toronto circle works).

City info Page

After selecting a specific city by clicking the circle at the bottom of homepage, the page will jump to a city info page, whcih shows a brief introduction of the city the user selected.

Restaurant page

Having the same side bar as the Toronto page, the same function. If we click Overview button, we will go to the Toronto page. If we click Restaurants button, we will stay on the current page.

This page has a list of restaurants with information, such as name, address, contact information, picture, has been there of want to go there. Whatâ€™s more, you can rating.

Restaurant name page

Having the same side bar as the Toronto page, the same function. If we click Overview button, we will go to the Toronto page. If we click Restaurants button, we will go to the restaurant page, which is the main page of restaurants(only La Carnita works right now).

When we click the restaurant La Carnita, it will go to this La Carnita page with some detail information about this restaurant. We can write comments about this restaurant. We also can do rating, and click has been there and want go buttons to provide some information, which will work later(comments cannot be submit yet).

Member info Page

Clicking on circle Vancouver, it will jump to Member info Page.

This page is a template, which is used to record member profile with some basic information. This jdo13's profile is just a sample, and we do not have real content yet. We can use the add to frinds button to add some friends. We can also send message to this person by clicking send message button.These two buttons are also templates, and we will work on it later.

Instructions

We use codeigniter and bootstrap for this project.

Images are stored in the image folder.

All other files are in the application folder.

Page layouts are in the views folder within application.

There are also some files for some navigation features in controllers.


To run in a webserver on CDF, simple enter the command: 'php -S localhost:port -t path/to/rootdir'
